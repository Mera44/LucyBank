# LucyBank

Steps To Run The Project

1. 	Import Project as Maven Project
2.	Create Database in MySql with name lucybank
3.	Go to application-context change <prop key="hibernate.hbm2ddl.auto">create-drop</prop> for the firstime Running 	the project. After Run you can change it to update.
